# DNT Dhaka Frontend

A polished dental clinic frontend focused on trust, conversion, and SEO-ready content.

## Features
- Modern multi-page React UI with clean Tailwind styling
- Conversion-focused sections for appointments, services, doctors, and testimonials
- SEO component integration for better search visibility
- Responsive layout optimized for desktop and mobile experiences
- Structured project organization ready for production deployment

## Tech Stack
- React
- Tailwind CSS
- Vite
